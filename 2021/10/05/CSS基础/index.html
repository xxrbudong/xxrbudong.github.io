CSS基础

CSS简介

CSS-网页的美容师

CSS是层叠样式表的简称。

CSS也是一种标记语言。

CSS主要设置HTML文本内容，图片的外形，yi’ji版面布局的外观样式。

CSS最大的价值：由HTML专注去做结构实现，样式交给CSS。

CSS语法规范

CSS规则由两部分组成：选择器以及一条或多条说明。

选择器适用于指定CSS的样式的HTML标签

属性和属性值之间键值对存在，有；结束，属性与属性值之间用: 分隔开。

CSS代码风格

开发书写方式

  1. 样式格式书写

    1. 紧凑格式
    2. 展开格式

  2. 样式大小写

    全部用小写字母书写

  3. 空格规范

    1. 属性值前面，冒号后面，保留一个空格
    2. 标签和{}之间也要保留空格

CSS基础选择器

选择器

选择器的作用

简单来说，就是选择标签用的。

CSS做了两件事，选对人，做对事。

选择器的分类

分为基础选择器和复合选择器两大类。

基础选择器由单个选择器组成的。

基础选择器又包括标签选择器、类选择器、id选择器、和通配符选择器。

标签选择器

就是标签名作为选择器。

会把页面中全部的同类标签选择到，不能对其中某一个进行单独设置，要改就全改。

类选择器（最常使用）

差异化选择不同的标签，单独选一个或者某几个标签，可以使用类选择器。

结构需要调用class属性来调用class类的意思。

注意

  1. 类选择器.后面紧跟类的名字。
  2. 不能用标签的名字做我们的类名。
  3. 长名词可以用短横线进行分割。
  4. 不要出现中文和数字在名字中。
  5. 命名规范。
