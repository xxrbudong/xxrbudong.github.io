CSS基础

CSS简介

CSS-网页的美容师

CSS是层叠样式表的简称。

CSS也是一种标记语言。

CSS主要设置HTML文本内容，图片的外形，yi’ji版面布局的外观样式。

CSS最大的价值：由HTML专注去做结构实现，样式交给CSS。

CSS语法规范

CSS规则由两部分组成：选择器以及一条或多条说明。

选择器适用于指定CSS的样式的HTML标签

属性和属性值之间键值对存在，有；结束，属性与属性值之间用: 分隔开。

CSS代码风格

开发书写方式

  1. 样式格式书写

    1. 紧凑格式
    2. 展开格式

  2. 样式大小写

    全部用小写字母书写

  3. 空格规范

    1. 属性值前面，冒号后面，保留一个空格
    2. 标签和{}之间也要保留空格

CSS基础选择器

选择器

选择器的作用

简单来说，就是选择标签用的。

CSS做了两件事，选对人，做对事。

选择器的分类

分为基础选择器和复合选择器两大类。

基础选择器由单个选择器组成的。

基础选择器又包括标签选择器、类选择器、id选择器、和通配符选择器。

标签选择器

就是标签名作为选择器。

会把页面中全部的同类标签选择到，不能对其中某一个进行单独设置，要改就全改。

类选择器（最常使用）

差异化选择不同的标签，单独选一个或者某几个标签，可以使用类选择器。

结构需要调用class属性来调用class类的意思。

注意

  1. 类选择器.后面紧跟类的名字。
  2. 不能用标签的名字做我们的类名。
  3. 长名词可以用短横线进行分割。
  4. 不要出现中文和数字在名字中。
  5. 命名规范。
多类名

我们可以给一个标签指定多个类名。

使用方式：class="red font20"

可以把相同的样式，放入到同一个类中。

id选择器

只能调用一次，别的标签切勿使用。

通配符选择器

*会把所有的标签选出来，会把所有的标签更改某种样式。

通配符选择器不需要调用，自动给所有元素使用样式。

CSS字体属性

字体序列

CSS使用font-family属性定义文本字体序列。

各种字体之间必须使用英文状态下逗号隔开，多种字体是为了保证字体的正常显示。

如果多个单词组成，那么空格隔开用引号引起来

字体大小

使用font-size来指定字体大小。

谷歌默认为16px。

标题标签比较特殊，需要单独指定文字大小。

字体粗细

font-weight来设置字体粗细。

400 相当于normal，700相当于bold。

文本样式

font-style设置文本的风格。

分为normal和italic（倾斜）

字体复合属性

不需要设置的属性可以省略，不能更换顺序，必须保留font-size和font-family。

CSS文本属性

文本颜色

color: 定义文本的颜色。

预定义的颜色值，十六进制表示颜色，RGB（三原色表示法）代码格式。

对齐文本

text-align属性用于设置文本内容的水平对齐方式。

left center right。

装饰文本

text-decoration 规定文本的修饰。给文本增加下划线，删除线，上划线等。

属性值 描述
none 默认
underline 下划线
overline 上划线
line-through 删除线

文本缩进

text-indent第一行的缩进，段落的首行进行缩进。

em单位，是一个相对单位，以当前元素一个文字的大小。

行间距

line-height 设置行间距。

行间距包括：上间距，文本高度，下间距。
